# Pou Navidad — Juego en p5.js

Este es un juego estilo endless runner / flappy, desarrollado con p5.js, donde controlas a un personaje (Pou navideño) que debe esquivar paredes mientras el fondo se desplaza y suenan efectos de sonido festivos.

## Herramientas de IA utilizadas

Este proyecto hizo uso de herramientas de Inteligencia Artificial con fines educativos y de apoyo creativo:

Sora (sora.chatgpt.com) — utilizada para la generación de imágenes de referencia y elementos visuales conceptuales.
